# Code of Conduct

## Our Standard

This is a technical project handling financial security. We hold ourselves and contributors to a high standard:

- Be precise and honest — especially about security claims
- Disagree on technical merit, not personality
- Assume good faith in questions and reviews
- Protect user security above all else

## Unacceptable Behaviour

- False security claims
- Harassment of any kind
- Dismissing legitimate security concerns

## Enforcement

Violations reported to buildwithcolours@gmail.com are reviewed by the foundation board. Consequences range from warning to permanent ban depending on severity.
