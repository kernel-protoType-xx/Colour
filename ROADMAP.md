# Roadmap

- Core 16-layer cryptographic stack
- Bitcoin, Ethereum, Solana support
- MCP institutional deployment server
- GitHub release binaries (Windows, Mac, Linux)
- Full documentation and governing documents

## v0.2.0 — Audit

- Independent third-party security audit (Trail of Bits / NCC Group)
- Publish audit results
- Fix all findings
- Bug bounty programme live

## v0.3.0 — MPC

- Full threshold MPC for institutional deployments
- 2-of-3 and 3-of-5 distributed signing
- No complete key reconstruction anywhere

## v0.4.0 — Ecosystem

- Additional chain support (community governed)
- Formal verification of additional critical paths
- First foundation membership programme
- First certification issued

## v1.0.0 — Production

- Two independent audits completed
- Six months bug bounty with no critical findings
- First institutional deployment in production
- Sovereign deployment package
