# Colour Shield

**Post-quantum package security for npm, pip, and cargo.**

Colour Shield sits between you and your package manager. Every package
you install is verified before it touches your machine.

Built on Colour's post-quantum cryptographic infrastructure. Audited by Cure53.

## Install

\```bash
npm install -g colour-shield
\```

## Usage

\```bash
colour-shield npm install express
colour-shield pip install requests
colour-shield cargo add serde
colour-shield scan axios
colour-shield audit
colour-shield test
\```
